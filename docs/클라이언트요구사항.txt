NB High School Jobs – 홈페이지 제작 기획안 (Wix)
1. 제작 목적
이 사이트는 캐나다 뉴브런즈윅(New Brunswick) 주의 고등학생들이 방학 중 풀타임, 학기 중 주말 파트타임, 봉사활동(Volunteer) 등을 보다 쉽게 찾고, 기업은 고등학생 대상 일자리를 직접 구인공고로 등록할 수 있는 플랫폼을 제공하는 데 목적이 있습니다. 또한 레쥬메 업로드, 선생님 추천서 제출, 취업 설명회, 면접 스킬 교육 등 학생 진로 준비를 통합적으로 지원하는 구조로 구성됩니다.
2. 주요 사용자
- 학생(구직자): 레쥬메 업로드, 추천서 등록, 일자리 검색
- 기업/고용주(구인자): 구인공고 등록, 지원자 확인
- 선생님: 학생 추천서 업로드
- 관리자: 전체 콘텐츠 및 사용자 관리
3. 페이지 구성 (메뉴 구조)
• 홈(Home): 사이트 소개, 사용 방법 안내
• 학생 구직 페이지 (Job Seekers): 학생 레쥬메 업로드, 기본 정보 입력, 관리자 승인 후 게시
• 추천서 업로드 (Teacher References): 선생님 전용 추천서 업로드 페이지
• 구인 게시판 (Job Postings): 회사가 직접 구인공고 작성 및 등록
• 이벤트 안내 페이지:
  - 여름방학 취업설명회
  - 면접 스킬 워크숍
• 문의하기 (Contact): 사용자 문의 양식
4. 핵심 기능 요약
기능	설명
레쥬메 업로드 기능	학생이 자신의 이력서를 업로드할 수 있는 기능
추천서 업로드 기능	선생님이 추천서를 올릴 수 있는 전용 업로드 페이지
구인공고 등록 기능	회사가 직접 구인공고를 작성 및 제출 가능
관리자 승인 기능	업로드된 문서 및 공고는 관리자 승인 후 게시
설명회/워크숍 신청	이벤트 신청 기능 (Google Forms 또는 Wix 자체 폼 활용 가능)
모바일 최적화	모바일에서도 쉽게 이용 가능하게 반응형 디자인 필요
5. 디자인 요청 사항
- 청소년 친화적이고 신뢰감 있는 느낌의 디자인
- 메인 컬러: 파랑, 하늘색 계열
- 직관적인 메뉴 구조와 큰 버튼 사용
- 각 항목별 아이콘 삽입 가능
6. 관리자 접근 권한
- 게시물 승인/삭제
- 업로드된 문서 관리
- 사이트 내 각종 콘텐츠 편집 가능
- 폼 응답 확인 및 다운로드 가능
7. 기술 및 플랫폼
- Wix 기반 제작 필수
- 도메인: nbhighschooljobs.com
- 폼 기능, 업로드 기능, 이메일 자동연동 기능 포함
- 추가적으로 회원가입 기능은 선택 가능 (권한 설정 기능 활용)
✅ 마무리 요청
이 기획안에 따라 학생 레쥬메/추천서 업로드, 구인공고 등록, 행사 홍보, 고등학생 맞춤형 일자리 중개 기능이 포함된 홈페이지를 Wix 기반으로 제작해 주시기 바랍니다. 사이트는 영어로 제작하며, 추후 관리자가 직접 수정 가능하도록 설명도 함께 부탁드립니다.
