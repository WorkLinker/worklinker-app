🌐 Wix 도메인 → Vercel 연결 상세 가이드
작성일: 2025년 1월 15일
===============================================

📋 개요
고객이 Wix에서 구매한 도메인을 현재 Vercel에 배포된 
뉴브런즈윅 학생 취업 플랫폼에 연결하는 방법

===============================================
🔧 사전 준비사항
===============================================

✅ 필요한 정보:
- Wix 계정 로그인 정보
- 구매한 도메인명 (예: example.com)
- Vercel 프로젝트 접근 권한
- 현재 Vercel 배포 URL

✅ 확인 사항:
- Vercel 프로젝트가 정상 배포되어 있는지 확인
- 도메인이 Wix에서 활성화되어 있는지 확인
- 기존 웹사이트가 해당 도메인에 연결되어 있다면 백업

===============================================
📖 단계별 연결 방법
===============================================

🔸 1단계: Vercel에서 도메인 추가
-------------------------------------------
1. Vercel 대시보드 접속 (vercel.com)
2. 프로젝트 선택 (뉴브런즈윅 학생 플랫폼)
3. "Settings" 탭 클릭
4. 좌측 메뉴에서 "Domains" 선택
5. "Add" 버튼 클릭
6. 도메인명 입력 (예: yourdomain.com)
7. "Add" 버튼 클릭하여 추가

🔸 2단계: DNS 레코드 정보 확인
-------------------------------------------
Vercel에서 다음 중 하나의 방법을 제시합니다:

방법 A - CNAME 레코드:
- Type: CNAME
- Name: www (또는 @)
- Value: cname.vercel-dns.com

방법 B - A 레코드:
- Type: A
- Name: @ (root domain)
- Value: 76.76.19.61

※ Vercel에서 제공하는 정확한 값을 사용하세요

🔸 3단계: Wix 도메인 관리 접속
-------------------------------------------
1. Wix 계정 로그인 (wix.com)
2. "My Account" → "Domains" 클릭
3. 연결할 도메인 선택
4. "Manage DNS" 또는 "Advanced DNS" 클릭

🔸 4단계: DNS 레코드 설정
-------------------------------------------
기존 레코드 확인:
- 기존 A 레코드나 CNAME 레코드가 있다면 삭제 또는 수정

새 레코드 추가:
1. "Add Record" 버튼 클릭
2. Vercel에서 제공한 정보 입력:
   - Type: A 또는 CNAME
   - Host/Name: @ 또는 www
   - Points to/Value: Vercel 제공 값
3. TTL: 기본값 (보통 1시간)
4. "Save" 또는 "Apply" 클릭

🔸 5단계: SSL 및 www 설정
-------------------------------------------
Vercel에서 추가 설정:
1. www.yourdomain.com도 함께 추가
2. "Redirect www to domain" 또는 그 반대 설정
3. SSL 인증서 자동 발급 대기 (24-48시간)

===============================================
⏰ 타임라인 및 예상 소요시간
===============================================

즉시 완료: Vercel 도메인 추가 (5분)
즉시 완료: Wix DNS 설정 (10분)
1-4시간: DNS 전파 시작
24-48시간: 완전한 DNS 전파 및 SSL 발급
최대 72시간: 모든 지역에서 접속 가능

===============================================
🔍 연결 상태 확인 방법
===============================================

DNS 전파 확인:
- whatsmydns.net 사이트에서 도메인 입력
- 전 세계 DNS 서버에서 전파 상태 확인

SSL 인증서 확인:
- https://yourdomain.com 접속
- 브라우저 주소창에 자물쇠 아이콘 확인

기능 테스트:
- 홈페이지 정상 로딩 확인
- 로그인/회원가입 기능 테스트
- 관리자 페이지 접속 확인

===============================================
⚠️ 주의사항 및 문제 해결
===============================================

🚨 일반적인 문제들:

1. "Domain not found" 오류
원인: DNS 전파 대기 중
해결: 24-48시간 기다리기

2. "SSL Certificate Error"
원인: 인증서 발급 대기 중
해결: Vercel에서 SSL 재발급 요청

3. "502 Bad Gateway"
원인: 서버 설정 문제
해결: Vercel 배포 상태 확인

4. "Redirect Loop"
원인: www 리다이렉트 설정 충돌
해결: Vercel에서 리다이렉트 설정 수정

🛠️ 긴급 복구 방법:
문제 발생 시 Wix DNS 설정을 원래대로 되돌리고
기존 Vercel URL로 임시 접속 가능
