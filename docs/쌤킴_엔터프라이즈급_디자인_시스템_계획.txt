7. **홀로그래픽 효과** - 반짝이는 3D 느낌, 미래적 분위기

## 🏥 업종별 디자인 DNA 시스템

### 한방한의원 전용 디자인:
```
✓ 색상: 2025 Mocha Mousse + 자연 그린 조합
✓ 타이포그래피: 신뢰감 있는 대형 헤드라인
✓ 레이아웃: 전통과 현대의 조화
✓ 애니메이션: 부드럽고 치유적인 느낌
✓ 컴포넌트: 예약 시스템, 의료진 카드, 치료 안내
```

### 기타 업종 디자인 DNA:
```
🏢 기업/컨설팅 → 프리미엄 블루 + 미니멀 + 강렬한 임팩트
🍴 레스토랑 → 따뜻한 컬러 + 음식 사진 + 감성적 스토리텔링
🎓 교육기관 → 밝은 색상 + 친근함 + 인터랙티브 요소
🏠 부동산 → 신뢰감 + 고급스러움 + 가상 투어
💄 뷰티 → 소프트 그라디언트 + 엘레간트 + 제품 쇼케이스
⚖️ 법무법인 → 깊은 네이비 + 금색 포인트 + 전문성 강조
☕ 카페 → 따뜻한 브라운 + 아늑함 + 커피 향이 느껴지는 UI
🏠 펜션 → 자연 그린 + 휴식 테마 + 풍경 중심 갤러리
🦷 치과 → 깔끔한 화이트 + 민트 포인트 + 청결함 강조
```
