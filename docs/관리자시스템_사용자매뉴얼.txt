        📚 관리자 시스템 사용자 매뉴얼
===================================================================

📅 최종 업데이트: 2024년 12월
🎯 대상 사용자: 시스템 관리자
📋 플랫폼: 캐나다 뉴브런즈윅 고등학생 구직 플랫폼

===================================================================
        🚀 시작하기
===================================================================

🔑 [1] 관리자 계정 로그인
   1. 홈페이지 상단의 "로그인" 버튼 클릭
   2. 관리자 이메일과 비밀번호 입력
   3. 로그인 성공 후 자동으로 관리자 페이지로 이동
   
   ⚠️ 주의사항:
   - 관리자 권한이 없는 계정으로는 접근 불가
   - 잘못된 접근 시 홈페이지로 자동 리다이렉트

🖥️ [2] 관리자 대시보드 개요
   📊 5개 주요 탭:
   - 👥 사용자 승인/거절: 구직 신청 검토
   - ✏️ 콘텐츠 수정: 홈페이지 텍스트 관리
   - 📋 활동 로그 열람: 관리자 활동 추적
   - 📁 로그 내보내기: 데이터 내보내기 안내
   - ⚙️ 관리자 설정: 계정 및 시스템 설정

===================================================================
        👥 [탭 1] 사용자 승인/거절
===================================================================

📋 기능 개요:
   구직을 신청한 고등학생들의 신청서를 검토하고 승인 또는 거절 처리

🔍 [1] 신청서 검색 및 필터링
   📌 검색 방법:
   - 검색창에 이름, 이메일, 학교명 입력
   - 실시간 검색 결과 표시
   - 대소문자 구분 없이 부분 검색 가능

📊 [2] 신청서 정보 확인
   ✅ 표시되는 정보:
   - 👤 신청자 이름
   - 📧 이메일 주소
   - 🏫 학교명 및 학년
   - 📞 연락처
   - 📄 이력서 파일명 (업로드한 경우)
   - 💼 기술/경험 내용
   - 📅 신청일

⚡ [3] 승인/거절 처리
   ✅ 승인 절차:
   1. 신청서 내용 검토
   2. "승인" 버튼 클릭
   3. 확인 팝업에서 "확인" 클릭
   4. 승인 완료 메시지 확인
   5. 해당 신청서가 목록에서 자동 제거

   ❌ 거절 절차:
   1. "거절" 버튼 클릭
   2. 거절 사유 입력 (선택사항)
   3. 확인 후 거절 처리 완료
   4. 해당 신청서가 목록에서 자동 제거

💡 처리 팁:
   - 모든 승인/거절 활동은 자동으로 로그에 기록됨
   - 처리 중 상태가 표시되어 중복 클릭 방지
   - 거절 사유는 향후 통계 분석에 활용 가능

===================================================================
        ✏️ [탭 2] 콘텐츠 수정
===================================================================

📋 기능 개요:
   홈페이지의 텍스트 내용을 실시간으로 수정하고 즉시 반영

🎯 [1] 히어로 슬라이드 편집
   📝 수정 가능한 내용:
   - 슬라이드 1, 2, 3의 제목과 부제목
   - 각 슬라이드는 독립적으로 편집 가능
   
   💡 작성 팁:
   - 제목: 임팩트 있는 핵심 메시지 (20자 이내 권장)
   - 부제목: 구체적인 설명이나 행동 유도 (30자 이내 권장)

🔘 [2] CTA 버튼 텍스트
   - 메인 페이지 "지금 시작하기" 버튼 텍스트 수정
   - 행동을 유도하는 명확한 문구 사용 권장

📄 [3] 메인 섹션 편집
   🏷️ 배지 텍스트: 플랫폼의 특징이나 지역성 강조
   📰 제목: 서비스의 핵심 가치 제안
   🔹 부제목: 감정적 어필이나 비전 제시
   📝 설명 텍스트: 상세한 서비스 소개 (2-3문장)
   ⭐ 강조 텍스트: 핵심 기능이나 차별점 부각

🎴 [4] 기능 카드 편집 (4개)
   🎓 학생 구직 카드:
   - 제목, 설명, 버튼 텍스트 수정 가능
   - 고등학생 대상 구직 서비스 소개
   
   📋 추천서 지원 카드:
   - 추천서 관련 서비스 설명
   - 신뢰도 및 전문성 강조
   
   🏢 기업 채용 카드:
   - 기업 대상 채용 서비스 소개
   - 효율성 및 매칭 품질 강조
   
   🎉 교육 이벤트 카드:
   - 교육 프로그램 및 이벤트 안내
   - 참여 혜택 및 성장 기회 제시

💾 [5] 변경사항 저장
   📌 저장 절차:
   1. 모든 수정 내용 입력 완료
   2. 하단의 "변경사항 저장" 버튼 클릭
   3. 로딩 상태 확인 (저장 중... 표시)
   4. 성공 메시지 확인
   5. 홈페이지에서 즉시 변경 내용 확인 가능

⚠️ 주의사항:
   - 저장하기 전까지는 변경사항이 반영되지 않음
   - 저장 시 모든 관리자 활동이 로그에 기록됨
   - 변경 전/후 내용이 비교 저장되어 복원 가능

===================================================================
        📋 [탭 3] 활동 로그 열람
===================================================================

📋 기능 개요:
   모든 관리자 활동을 추적하고 검색, 필터링, 내보내기 가능

🔍 [1] 로그 필터링 시스템
   📅 기간 선택 (드롭다운):
   - 🗓️ 전체 기간: 모든 로그 표시
   - 📅 오늘: 오늘 날짜만
   - 📆 어제: 어제 날짜만
   - 📊 최근 7일: 일주일 전부터 오늘까지
   - 📈 최근 30일: 한 달 전부터 오늘까지
   - ⚙️ 직접 선택: 커스텀 날짜 범위 설정

   👤 관리자 이메일 검색:
   - 부분 검색 지원 (대소문자 무시)
   - 특정 관리자의 활동만 필터링

   🏷️ 작업 유형 필터:
   - 콘텐츠 수정: 홈페이지 텍스트 변경
   - 사용자 승인/거절: 구직 신청 처리
   - 시스템: 시스템 관련 작업
   - 관리자: 비밀번호 변경 등

📊 [2] 로그 테이블 정보
   🕐 작업시간: 정확한 날짜와 시간 (초 단위)
   👨‍💼 관리자 이메일: 작업 수행한 관리자
   🏷️ 작업 유형: 색상별 뱃지로 구분
   📝 작업 내용: 수행된 작업의 간단한 설명
   📄 상세 정보: 대상자, 사유, 변경사항 등

🔄 [3] 정렬 및 탐색
   ⬆️⬇️ 정렬 기능:
   - 테이블 헤더 클릭으로 오름차순/내림차순 전환
   - 기본: 최신 순서 (작업시간 내림차순)

   📖 페이지네이션:
   - 20개씩 표시 (성능 최적화)
   - 이전/다음 및 페이지 번호 클릭
   - 현재 범위 표시 ("1-20개 / 총 50개")

📁 [4] 로그 내보내기
   📄 PDF 내보내기:
   - 현재 표시된 로그 테이블을 PDF로 저장
   - 보고서 형태로 제목과 생성일 포함
   - 여러 페이지 자동 분할
   - 파일명: "활동로그_YYYY-MM-DD.pdf"

   📊 CSV 내보내기:
   - Excel에서 바로 열기 가능한 CSV 형식
   - 한글 인코딩 지원 (UTF-8 BOM)
   - 필터링된 데이터만 내보내기
   - 파일명: "활동로그_YYYY-MM-DD.csv"

💡 활용 팁:
   - 정기적인 감사 시 특정 기간으로 필터링 후 PDF 내보내기
   - 관리자별 활동 분석 시 이메일 필터 활용
   - 문제 발생 시 해당 시점 전후 로그 검토

===================================================================
        📁 [탭 4] 로그 내보내기
===================================================================

📋 현재 상태:
   활동 로그 열람 탭에서 PDF/CSV 내보내기 기능 이용 가능

📈 향후 추가 예정 기능:
   - 자동 리포트 생성 (일간/주간/월간)
   - 이메일 자동 발송
   - 데이터 분석 및 트렌드 차트
   - 커스텀 리포트 템플릿

🎯 현재 이용 방법:
   "활동 로그 열람" 탭 → PDF/CSV 내보내기 버튼 이용

===================================================================
        ⚙️ [탭 5] 관리자 설정
===================================================================

📋 기능 개요:
   관리자 계정 정보 확인 및 보안 설정 관리

👤 [1] 계정 정보 확인
   📧 현재 이메일: 로그인에 사용되는 관리자 이메일
   🕐 마지막 로그인: 최근 로그인 시간
   📅 계정 생성일: 관리자 계정이 생성된 날짜

🔐 [2] 비밀번호 변경
   🔑 변경 절차:
   1. "🔐 비밀번호 변경" 버튼 클릭
   2. 모달 창에서 현재 비밀번호 입력
   3. 새 비밀번호 입력 (보안 요구사항 준수)
   4. 새 비밀번호 확인 입력
   5. "변경하기" 버튼 클릭
   6. 성공 메시지 확인

   🔒 비밀번호 요구사항:
   - ✅ 최소 8자 이상
   - ✅ 대문자 포함 필수
   - ✅ 소문자 포함 필수
   - ✅ 숫자 포함 필수
   - ✅ 특수문자 (@$!%*?&) 포함 필수

   👁️ 편의 기능:
   - 각 입력 필드별 비밀번호 표시/숨김 토글
   - 실시간 유효성 검사 및 에러 표시
   - 요구사항 안내 패널

🛡️ [3] 보안 권장사항
   📋 정기적인 보안 관리:
   - 비밀번호 3개월마다 변경
   - 다른 사이트와 동일한 비밀번호 사용 금지
   - 강력한 비밀번호 조합 사용
   - 의심스러운 활동 발견 시 즉시 비밀번호 변경

💻 [4] 시스템 정보
   👨‍💼 관리자 권한: 전체 시스템 관리 권한
   ⏰ 세션 만료: 24시간 후 자동 로그아웃
   🔒 보안 수준: 높음 (Firebase Authentication 기반)

===================================================================
        🚨 문제 해결 가이드
===================================================================

❓ [1] 로그인 관련 문제
   🔧 로그인이 안 될 때:
   - 이메일과 비밀번호 정확성 확인
   - 관리자 권한 보유 여부 확인
   - 브라우저 캐시 및 쿠키 삭제 후 재시도
   - 다른 브라우저에서 시도

   🔧 관리자 권한 없음 오류:
   - 시스템 관리자에게 권한 요청
   - 올바른 관리자 계정으로 로그인 확인

❓ [2] 콘텐츠 수정 관련 문제
   🔧 변경사항이 저장되지 않을 때:
   - 네트워크 연결 상태 확인
   - 브라우저 새로고침 후 재시도
   - 필수 입력 필드 누락 여부 확인

   🔧 홈페이지에 반영되지 않을 때:
   - 브라우저 캐시 새로고침 (Ctrl+F5)
   - 다른 기기에서 확인
   - 시간 경과 후 재확인 (최대 1-2분)

❓ [3] 로그 및 내보내기 문제
   🔧 로그가 표시되지 않을 때:
   - 필터 설정 확인 및 초기화
   - 날짜 범위 설정 확인
   - 관리자 이메일 필터 초기화

   🔧 PDF/CSV 다운로드 실패:
   - 브라우저 팝업 차단 해제
   - 다운로드 폴더 권한 확인
   - 파일명에 특수문자 제거

❓ [4] 비밀번호 변경 문제
   🔧 현재 비밀번호 오류:
   - 정확한 현재 비밀번호 입력 확인
   - Caps Lock 상태 확인
   - 복사/붙여넣기 시 공백 확인

   🔧 새 비밀번호 요구사항 오류:
   - 모든 요구사항 준수 확인
   - 특수문자 정확한 문자 사용
   - 비밀번호 확인 일치 여부 점검
